# 精霊義伝 〜108の神器〜

このリポジトリは、連載小説
**「精霊義伝 〜108の神器〜」** を
Zennのブック形式で公開・販売するため事を検討中のものです。

ここに収録されている文章は、
すでに完成した本文のみで構成されています。

---

## この物語について

精霊と人が、同じ世界に生きている。

それは祝福ではなく、
均衡でもなく、
ただ、そうであるという事実として存在している。

この物語は、
世界の仕組みを説明するためのものではありません。

名付けられる前の感情、
言葉になる前の違和感、
選択の直前に訪れる沈黙。

それらを、一人の語り手の内側から辿っていきます。

---

## 構成について

本作は、以下の単位で構成されています。

- 惑星編
- シーズン（全12話）
- 1話＝1章

Zennでは
**「1シーズン＝1ブック」** として公開しています。

各ブックは、そのシーズンの物語として完結しますが、
世界全体の謎は、静かに次へと引き継がれていきます。

---

## 読み方のおすすめ

- 順番通りに読むことを推奨します
- 途中で立ち止まり、戻っても構いません
- すべてを理解しようとしなくて大丈夫です

この物語は、
理解よりも「残る感覚」を大切にしています。

---

## 無料／有料について

各ブックには、

- 無料で読める導入部分
- 有料で読める本文

が含まれています。

無料部分は、
この世界の呼吸に触れるための入口です。

続きを読むかどうかは、
その感覚に身を委ねて決めてください。

---

## 最後に

このリポジトリには、

- 設定資料
- 解説
- 裏話

は含まれていません。

ここにあるのは、
物語そのものだけです。

それが、あなたの中で
どのような形をとるのか。

その答えは、
物語の外にあります。
